# Data directory

Do not commit raw or processed data files to this repository.

Expected local layout after obtaining the source datasets:

```text
data/
  nhanes/
    processed/
      nhanes_2005_2018_diabetes_ckd_mortality_scan.csv
  mimic/
    processed/
      generated by the MIMIC preprocessing scripts
```

MIMIC-IV source files should either be placed outside the repository and exposed
through the `MIMIC_ROOT` environment variable, or placed locally under
`data/mimic-iv-3.1/` by a credentialed user. MIMIC-IV source or processed files
must not be committed.
